# Radiology Authorization System

## Overview

This project is a healthcare decision support system for determining radiology imaging authorization requirements.

## Features (Planned)

- Authorization rules by insurance and modality
- Scheduling decision support
- AI-based explanation of rules
- PDF guideline parsing (future)

## Tech Stack

- Backend: Spring Boot
- Database: MySQL
- Frontend: React.js

## Project Structure

- backend → Spring Boot API
- frontend → React UI
- database → SQL scripts
